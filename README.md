# Kubernetes Practice with Nginx

This repository contains my practice exercises for deploying and managing **Nginx** on **Kubernetes**.  
The goal is to strengthen my understanding of Pods, Deployments, Services, and other Kubernetes concepts by working with a simple, widely used web server.
